# DC_PC_Controller

# 🤖 Discord System Monitor Bot

Gelişmiş bir Discord botu ile bilgisayarınızın veya sunucunuzun sistem durumunu **gerçek zamanlı** olarak izleyin.

Bu bot, CPU, RAM, disk kullanımı gibi kritik bilgileri analiz eder ve Discord üzerinden canlı olarak raporlar.

---

## 🚀 Özellikler

- 🔥 CPU kullanım takibi
- 💾 RAM kullanım analizi
- 📀 Disk kullanım durumu
- 📡 Network (Upload / Download) izleme
- ⏱️ Sistem çalışma süresi (Uptime)
- 🧠 İşletim sistemi bilgisi
- 🏓 Bot gecikmesi (Ping)
- 🔄 Canlı mesaj güncelleme (real-time tracking)
- 🚨 Yüksek kullanım uyarı sistemi

---

## 🛠️ Kullanılan Teknolojiler

- Python 3
- discord.py
- psutil
- platform (built-in)
